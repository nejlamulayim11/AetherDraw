# AetherDraw

AetherDraw is a real-time computer vision application that allows users to draw and write in the air using hand tracking.

## Features

- Real-time hand tracking
- Air drawing
- Gesture recognition
- Virtual canvas
- Shape recognition (planned)
- OCR (planned)

## Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy
